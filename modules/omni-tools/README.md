# Omni-Tools for BioDockify

This module runs a local instance of [Omni-Tools](https://github.com/iib0011/omni-tools), a collection of privacy-first web utilities (PDF tools, image editing, etc.).

## Quick Start
1. Open this directory in terminal.
2. Run `docker-compose up -d`.
3. Tools will be available at `http://localhost:8081`.

## Integration
BioDockify embeds this service in the "Tools" tab for seamless access.
